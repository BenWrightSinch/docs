SINVideoFrameCallback Protocol Reference

Conforms to NSObject
Declared in SINVideoFrameCallback.h

Overview

The callback object that will process frames from a remote stream.

– onFrame:callId: required method

This method is called when a new frame is received.

- (void)onFrame:(id<SINVideoFrame>)videoFrame callId:(NSString *)callId

Parameters

callId

The identifier of the call that received a frame.
frame

The video frame.

Discussion

IMPORTANT: The implementor of this protocol is responsible for explicitly releasing the frame by calling [SINVideoFrame releaseFrame].
